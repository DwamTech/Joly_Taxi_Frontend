# صفحة الخريطة المباشرة

## الوصف
صفحة لمراقبة السائقين والرحلات على الخريطة في الوقت الفعلي.

## المكونات

### 1. LiveMapHero
- عرض الإحصائيات الرئيسية
- عدد السائقين المتصلين
- عدد الرحلات الجارية
- عدد الطلبات المفتوحة
- عدد السائقين المتاحين

### 2. MapFilters
- عرض/إخفاء السائقين
- عرض/إخفاء الرحلات
- فلترة حسب الحالة (متاح، في رحلة، مشغول)
- فلترة حسب نوع المركبة

### 3. MapView
- خريطة تفاعلية (محاكاة)
- عرض السائقين بأيقونات حسب نوع المركبة
- ألوان حسب الحالة:
  - 🟢 أخضر: متاح
  - 🟡 أصفر: في رحلة
  - 🔴 أحمر: مشغول
- معلومات عند النقر على السائق:
  - الاسم
  - نوع المركبة
  - الحالة
  - التقييم
  - رحلات اليوم
  - رقم الهاتف
- عرض علامات الرحلات الجارية

### 4. MapSidebar
- **إحصائيات سريعة:**
  - آخر تحديث (تلقائي كل 10 ثواني)

- **قائمة السائقين المتصلين:**
  - الاسم
  - نوع المركبة
  - التقييم
  - الحالة

- **قائمة الرحلات الجارية:**
  - رقم الرحلة
  - السائق
  - الراكب
  - من/إلى
  - الوقت المنقضي
  - المسافة
  - السعر
