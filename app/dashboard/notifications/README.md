# صفحة إدارة الإشعارات

## الوصف
صفحة شاملة لإدارة وإرسال الإشعارات للمستخدمين والسائقين في نظام جولي تاكسي.

## المكونات الرئيسية

### 1. NotificationsHero
عرض إحصائيات الإشعارات:
- إجمالي الإشعارات المرسلة
- إشعارات اليوم
- الإشعارات الفاشلة

### 2. SendNotificationForm
نموذج إرسال إشعار جديد يتضمن:
- اختيار نوع المستلمين (الجميع، السائقين، الركاب، مستخدم محدد، مجموعة مخصصة)
- محتوى الإشعار (عربي/إنجليزي)
- نوع الإشعار (معلومة، تحذير، عاجل)
- قنوات الإرسال (قاعدة البيانات، Push، SMS، Email)
- خيارات الإرسال (فوري أو مجدول)

### 3. NotificationsHistory
سجل الإشعارات المرسلة مع:
- جدول عرض تفصيلي
- فلاتر البحث والتصفية
- إجراءات (عرض، إعادة إرسال، حذف)

### 4. NotificationTemplates
إدارة قوالب الإشعارات:
- عرض القوالب المحفوظة
- إضافة قالب جديد
- تعديل القوالب الموجودة
- حذف القوالب
- استخدام القالب للإرسال

## البيانات
البيانات الوهمية موجودة في: `data/notifications/notifications-data.json`

## المسار
`/dashboard/notifications`

## التصميم
تصميم عصري واحترافي مع:
- ألوان متدرجة جذابة
- تأثيرات حركية سلسة
- تصميم متجاوب لجميع الشاشات
- واجهة سهلة الاستخدام
