# صفحة إدارة المستخدمين

## نظرة عامة
صفحة شاملة لإدارة جميع المستخدمين في النظام (سائقين، ركاب، كلاهما، إداريين).

## المكونات

### 1. UsersFilters
مكون الفلاتر والبحث:
- البحث بالاسم
- البحث برقم الهاتف
- فلتر النوع (الكل، راكب فقط، سائق فقط، كلاهما فقط)
- فلتر الحالة (نشط، محظور)
- الترتيب (الأحدث، الأقدم، الأكثر نشاطاً)

**ملاحظة مهمة عن فلتر النوع:**
- "راكب فقط": يعرض المستخدمين من نوع `user` فقط (لا يشمل `both`)
- "سائق فقط": يعرض المستخدمين من نوع `driver` فقط (لا يشمل `both`)
- "كلاهما فقط": يعرض المستخدمين من نوع `both` فقط
- "الكل": يعرض جميع الأنواع

### 2. UsersTable
جدول عرض المستخدمين:
- الاسم مع الصورة الرمزية
- رقم الهاتف
- النوع (راكب/سائق/كلاهما/إداري)
- الحالة (نشط/محظور)
- تاريخ التسجيل
- آخر نشاط
- الإجراءات:
  - عرض التفاصيل
  - حظر/إلغاء الحظر
  - إرسال إشعار
  - حذف

### 3. UserDetailsModal
نافذة تفاصيل المستخدم الكاملة:

#### معلومات مشتركة لجميع المستخدمين:
- الاسم الكامل
- رقم الهاتف
- البريد الإلكتروني
- كود الوكيل
- رقم المندوب
- الحالة
- تاريخ التسجيل
- آخر تسجيل دخول

#### معلومات إضافية للسائقين:
- رقم الهوية
- تاريخ انتهاء رخصة القيادة
- تاريخ انتهاء صلاحية الملف
- حالة التحقق (قابلة للتعديل من القائمة المنسدلة)
- الحالة الحالية (Online/Offline) - يتم تحديثها لحظياً
- متوسط التقييم وعدد التقييمات
- عدد الرحلات المكتملة والملغاة
- معلومات المركبة الكاملة
- جدول الاشتراكات (الاشتراك الحالي مميز باللون)
- المستندات (صورة الهوية، رخصة القيادة، رخصة المركبة)

#### معلومات إضافية للركاب:
- متوسط التقييم وعدد التقييمات
- عدد الرحلات المكتملة والملغاة

#### أزرار الإجراءات:
- سجل النشاط (آخر 10 رحلات)
- الشكاوى (تبويبين: شكاوى مقدمة ضده، شكاوى مقدمة منه)
