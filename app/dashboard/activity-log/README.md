# صفحة سجل النشاطات

## الوصف
صفحة شاملة لتتبع جميع الإجراءات الإدارية في لوحة التحكم.

## المكونات

### 1. ActivityLogHero
- عرض إحصائيات النشاطات
- إجمالي النشاطات
- نشاطات اليوم
- المسؤولين النشطين
- الإجراءات الحرجة (حذف، حظر)

### 2. ActivityLogFilters
- البحث في الإجراءات والمسؤولين
- فلترة حسب نوع الإجراء (12 نوع)
- فلترة حسب المسؤول
- فلترة حسب التاريخ (من - إلى)
- عرض عدد النتائج
- زر مسح الفلاتر

### 3. ActivityLogTable
جدول يعرض:
- المسؤول (اسم مع صورة رمزية)
- الإجراء (مع أيقونة ولون مميز)
- الكيان المتأثر (نوع + ID)
- التفاصيل
- عنوان IP
- التاريخ والوقت

## أنواع الإجراءات

### إجراءات النظام
- 🔓 تسجيل دخول (login)
- 🔒 تسجيل خروج (logout)

### إجراءات البيانات
- ➕ إضافة (create)
- ✏️ تعديل (update)
- 🗑️ حذف (delete)

### إجراءات المستخدمين
- 🚫 حظر (block)
- ✅ إلغاء حظر (unblock)

### إجراءات الاشتراكات
- ✔️ تفعيل (approve)
- ❌ رفض (reject)

### إجراءات أخرى
- 📤 إرسال (send)
- ⚙️ إعدادات (settings)

## الكيانات المتأثرة
- user - مستخدم
- subscription - اشتراك
- vehicle_type - نوع مركبة
- report - بلاغ
- notification - إشعار
- settings - إعدادات
- system - النظام

## الفلترة
- البحث النصي في: اسم المسؤول، الإجراء، التفاصيل
- فلترة حسب نوع الإجراء
- فلترة حسب المسؤول المحدد
- فلترة حسب نطاق التاريخ

## التصميم
- استخدام CustomSelect للقوائم المنسدلة
- تصميم احترافي مع ألوان مميزة
- جدول responsive مع card view للموبايل
- تنسيق التاريخ بالعربية
- عرض IP بخط monospace
