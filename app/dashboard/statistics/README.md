# صفحة التقارير والإحصائيات

## الوصف
صفحة شاملة لعرض جميع التقارير والإحصائيات الخاصة بالمنصة مع إمكانية التصدير والجدولة.

## المكونات

### 1. StatisticsHero
- عرض الإحصائيات الرئيسية في الهيرو
- إجمالي الرحلات، الإيرادات، السائقين النشطين، الركاب النشطين

### 2. TripReports (تقارير الرحلات)
- **إحصائيات عامة:**
  - إجمالي الرحلات
  - الرحلات المكتملة والملغاة
  - معدل الإلغاء
  - متوسط المدة والمسافة والسعر
  - إجمالي الإيرادات

- **تقرير حسب نوع المركبة:**
  - عدد الرحلات لكل نوع
  - الإيرادات لكل نوع
  - متوسط السعر
  - النوع الأكثر طلباً

- **تقرير حسب الوقت:**
  - الرحلات حسب اليوم (7 أيام)
  - الرحلات حسب الشهر (12 شهر)

### 3. UserReports (تقارير المستخدمين)
- **تقرير السائقين:**
  - عدد السائقين (الكلي، النشطين، غير النشطين)
  - السائقين الأعلى تقييماً (Top 10)
  - السائقين الأكثر رحلات (Top 10)
  - السائقين بمعدل إلغاء عالي

- **تقرير الركاب:**
  - عدد الركاب (الكلي، النشطين، غير النشطين)
  - متوسط التقييم
  - الركاب الأعلى تقييماً (Top 10)
  - الركاب الأكثر رحلات (Top 10)
  - الركاب بمعدل إلغاء عالي

- **معدل نمو المستخدمين:**
  - عدد المستخدمين الجدد شهرياً (سائقين وركاب)

### 4. RevenueReports (تقارير الإيرادات)
- **الإيرادات الإجمالية:**
  - إيرادات الرحلات
  - إيرادات الاشتراكات
  - إجمالي الإيرادات

- **الإيرادات حسب الفترة:**
  - يومية
  - أسبوعية
  - شهرية
  - سنوية

- **الإيرادات حسب نوع المركبة:**
  - الإيرادات والنسبة المئوية لكل نوع

### 5. ExportSchedule (التصدير والجدولة)
- **تصدير التقارير:**
  - اختيار نوع التقرير
  - اختيار الصيغة (PDF أو Excel)
  - تحديد الفترة الزمنية

- **جدولة التقارير:**
  - اختيار نوع التقرير
  - اختيار التكرار (يومي، أسبوعي، شهري)
  - إدخال البريد الإلكتروني
