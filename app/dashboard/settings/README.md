# صفحة الإعدادات العامة

## الوصف
صفحة شاملة لإدارة جميع إعدادات التطبيق العامة والتحكم في الخيارات المختلفة.

## المكونات الرئيسية

### 1. SettingsHero
عرض عنوان الصفحة ووصفها بتصميم جذاب

### 2. GeneralSettings
إعدادات عامة تشمل:
- معلومات التطبيق (البريد، الهاتف، رقم الدعم)
- إعدادات الرحلات (وقت الانتظار، صلاحية العرض، العمولة، نطاق البحث)
- إعدادات التقييمات (الحد الأدنى، التقييمات السلبية، التقييمات الإجبارية)
- إعدادات الإلغاء (الحد الأقصى للإلغاءات)
- إعدادات الاشتراكات (الأسعار، المدة، التجديد التلقائي، التذكير)

### 3. SecuritySettings
إعدادات الأمان والخصوصية:
- إعدادات OTP (مدة الصلاحية، عدد المحاولات، مدة الحظر)
- سياسة الخصوصية
- شروط الاستخدام
- مدة الاحتفاظ بالبيانات

### 4. SystemSettings
إعدادات النظام والنسخ الاحتياطي:
- وضع الصيانة
- رسالة الصيانة
- النسخ الاحتياطي التلقائي
- تكرار النسخ الاحتياطي
- إنشاء/استرجاع/رفع نسخة احتياطية

## البيانات
البيانات الوهمية موجودة في: `data/settings/settings-data.json`

## المسار
`/dashboard/settings`

## التصميم
- تصميم عصري واحترافي
- تبويبات منظمة
- نماذج سهلة الاستخدام
- أزرار تبديل (Toggle) للخيارات
- تصميم متجاوب لجميع الشاشات

## الميزات
- حفظ الإعدادات لكل قسم على حدة
- إشعارات نجاح عند الحفظ
- إدارة النسخ الاحتياطي الكاملة
- وضع الصيانة للتطبيق
- تحكم كامل في جميع الإعدادات
