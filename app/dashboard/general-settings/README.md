# صفحة الإعدادات العامة

## الوصف
صفحة شاملة لإدارة جميع إعدادات التطبيق العامة (منفصلة عن إعدادات الحساب الشخصي).

## المسارات
- **الإعدادات العامة**: `/dashboard/general-settings`
- **إعدادات الحساب**: `/dashboard/settings`

## المكونات

### صفحة الإعدادات العامة
تحتوي على 3 تبويبات:

1. **الإعدادات العامة**
   - معلومات التطبيق
   - إعدادات الرحلات
   - إعدادات التقييمات
   - إعدادات الإلغاء
   - إعدادات الاشتراكات

2. **الأمان والخصوصية**
   - إعدادات OTP
   - سياسة الخصوصية
   - شروط الاستخدام
   - مدة الاحتفاظ بالبيانات

3. **النظام والنسخ الاحتياطي**
   - وضع الصيانة
   - النسخ الاحتياطي التلقائي
   - إدارة النسخ الاحتياطية

### صفحة إعدادات الحساب
صفحة بسيطة تحتوي على:
- معلومات الحساب الشخصي (الاسم، البريد، الهاتف)
- تغيير كلمة المرور

## البيانات
`data/settings/settings-data.json`

## التصميم
- تصميم عصري واحترافي
- هيرو بنفس ألوان الصفحة الرئيسية
- تبويبات منظمة
- نماذج سهلة الاستخدام
