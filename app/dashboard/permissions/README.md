# صفحة إدارة الصلاحيات

## الوصف
صفحة شاملة لإدارة المسؤولين وصلاحياتهم في لوحة التحكم.

## المكونات

### 1. PermissionsHero
- عرض إحصائيات المسؤولين
- إجمالي المسؤولين
- المسؤولين النشطين
- المسؤولين المعطلين
- المدراء العامين

### 2. PermissionsFilters
- البحث بالاسم
- البحث بالبريد الإلكتروني
- فلترة حسب الدور (مدير عام، مدير، مشرف)
- فلترة حسب الحالة (نشط، معطل)
- عرض عدد النتائج
- زر مسح الفلاتر

### 3. AdminsTable
جدول يعرض:
- الاسم مع صورة رمزية
- البريد الإلكتروني
- رقم الهاتف
- الدور (مع أيقونة ولون مميز)
- الحالة (نشط/معطل)
- تاريخ الإضافة
- آخر تسجيل دخول
- الإجراءات (تعديل، تعطيل/تفعيل، حذف)

### 4. AdminModal
نافذة منبثقة لإضافة/تعديل مسؤول:
- الاسم
- البريد الإلكتروني
- رقم الهاتف
- الدور (باستخدام CustomSelect)
- الحالة (باستخدام CustomSelect)

## الأدوار والصلاحيات

### مدير عام (Super Admin) ⭐
- صلاحيات كاملة على جميع الصفحات
- يمكنه إدارة الصلاحيات

### مدير (Admin) 👨‍💼
- صلاحيات على معظم الصفحات
- لا يمكنه إدارة الصلاحيات
- لا يمكنه الوصول للإعدادات العامة

### مشرف (Moderator) 👤
- صلاحيات محدودة
- الرحلات، التقييمات، البلاغات، الخريطة، الحظر
- لا يمكنه إدارة المستخدمين أو الإعدادات

## الإجراءات

### إضافة مسؤول جديد
1. الضغط على زر "إضافة مسؤول جديد"
2. ملء البيانات المطلوبة
3. اختيار الدور والحالة
4. حفظ

### تعديل مسؤول
1. الضغط على أيقونة التعديل ✏️
2. تعديل البيانات
3. حفظ التعديلات

### تعطيل/تفعيل مسؤول
- الضغط على أيقونة القفل 🔒/🔓
- يتم تغيير الحالة مباشرة

### حذف مسؤول
1. الضغط على أيقونة الحذف 🗑️
2. تأكيد الحذف
3. يتم حذف المسؤول نهائياً

## التصميم
- استخدام CustomSelect للقوائم المنسدلة
- تصميم احترافي مع تأثيرات hover
- ألوان مميزة لكل دور
- استجابة كاملة للشاشات المختلفة
